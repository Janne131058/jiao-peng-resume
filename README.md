# Jiao Peng — Personal Resume Site

A single-page personal portfolio / resume website built with **Next.js (App Router)**, **TypeScript**, **Tailwind CSS**, an animated **Three.js** particle-network background, and **Framer Motion** scroll/entrance animations.

## Sections

- **Hero** — name, title, tagline, call-to-action
- **About** — skills grouped by category
- **Experience** — timeline of work history
- **Projects** — selected infrastructure & software projects
- **Education**
- **Contact** — email & phone

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Editing content

All resume content lives in [`lib/data.ts`](lib/data.ts) — edit that one file to update text; no component changes needed.

## Deploy

Push to GitHub and import the repo at [vercel.com/new](https://vercel.com/new). Vercel auto-detects Next.js — no configuration required.
